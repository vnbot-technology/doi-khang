# DoiKhang Anime - Fighting Game

Anime fighting game with LAN/internet multiplayer.

## How to Play

1. Open `godot_project/` in Godot 4.3+
2. Press F5 to run
3. Choose Play Local for same-keyboard play, or Play LAN/Internet for multiplayer

## Controls
- Player 1: WASD (move), J (attack), K (special), L (ultimate), Shift (block)
- Player 2: Arrow keys (move), Numpad 1 (attack), Numpad 2 (special), Numpad 3 (ultimate), Numpad 0 (block)

## Characters
- Goku (Dragon Ball): Kamehameha beam, Super Saiyan transformation
- Naruto (Naruto): Rasengan dash, Shadow Clone summon
- Luffy (One Piece): Gum-Gum Pistol, Gear Second speed boost
- Conan (Detective Conan): Stun gun, Soccer ball combo
- Doraemon (Doraemon): Anywhere Door teleport, Bamboo Copter spin
- Sakura (Naruto): Healing jutsu, Cherry Blossom impact

## Relay Server
Start: `cd relay_server && npm install && node server.js`
Deploy to Railway/Render and update RELAY_URL in RelayClient.gd
